# Smarty
A React/Typescript web application to display a catalogue of Smartphones

| CI status | URL |
|-|-|
|![baez97](https://circleci.com/gh/baez97/smarty-webapp.svg?style=svg) | [production](https://smarty-webapp.herokuapp.com)

## The Home Screen
The app will display an horizontal list of the most relevant devices of our collection under the "Best Sellers" title, and a list of all the available smartphones showing name, a photo (when available) and the price.
